# Gestor Financiero

App para poder trackear los gastos personales de las personas.

## Features
- Ingreso de **gastos** 
    - Estos ingresos se pueden agrupar por categorias
- Gastos hechos
    - Estos gastos se pueden agrupar en categorias
- Grupos familiares
    - Trackeo de Ingresos/Gastos por miembro familiar
